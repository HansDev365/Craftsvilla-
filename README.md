# CraftsVilla

CraftsVilla is a multi-page website for a creative workshop business.  
The project presents art workshops for families, friends, and teams in a warm and inviting way.

## Project Overview

This website was created as a school project to practice front-end web development using HTML and CSS.  
The goal was to design a visually appealing and user-friendly website for a fictional workshop brand called **CraftsVilla**.

The site includes:
- a home page with a hero section and featured workshop kits
- a booking page
- a gallery page
- a feedback page with a review form and testimonials

## Features

- Responsive multi-page website
- Navigation bar shared across pages
- Hero section with call-to-action buttons
- Workshop cards with images, descriptions, and prices
- Feedback form with workshop selection
- Testimonial section for social proof
- Custom styling with reusable CSS classes
- Grid and flexbox layout
- Soft, creative visual design

## Pages

### Home
The home page introduces the brand and highlights the main message:
**Create. Connect. Unwind.**

### Feedback
The feedback page includes a form where visitors can share their experience, choose a workshop, and write about their favorite moment. It also includes example testimonials. 

### Styling
The website uses one shared stylesheet with reusable classes for layout, cards, buttons, forms, gallery items, and responsive behavior. 

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid

## Purpose of the Project

The purpose of this project was to:
- practice building a structured website with multiple pages
- improve HTML and CSS skills
- learn how to create reusable components and layouts
- design a website with a clear visual identity
- make the layout responsive for smaller screens

## What I Learned

Through this project, I practiced:
- creating page structure with semantic HTML
- styling layouts with Flexbox and Grid
- building forms and content cards
- improving visual hierarchy and spacing
- making a website more responsive and user-friendly
